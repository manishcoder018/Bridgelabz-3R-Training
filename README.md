# Backend-Dev # introduction_to_database

1. What are the advantages of using Mongoose over the native MongoDB driver?
Answer :- Schema-based structure → Defines clear data structure (schema)
Data validation → Built-in validation before saving data
Easy relationships → Supports population (like joins)
Middleware support → Pre & post hooks (e.g., before save)
Simplified queries → Cleaner and readable syntax
Model abstraction → Work with models instead of raw collections
Built-in methods → Custom methods & statics
Auto type casting → Converts data types automatically
